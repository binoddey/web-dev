Lets learn Lists in HTML

A list is a collection of items.

There are two types of lists:

  • Unordered List
  • Ordered List

Unordered List is a list where the items are not ordered.

  • Apple
  • Banana
  • Cherry

Ordered List is a list where the items are ordered.

  1. Apple
  2. Banana
  3. Cherry

Definition List is a list where the items are defined.
